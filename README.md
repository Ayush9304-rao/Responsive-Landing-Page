# Responsive Landing Page

A responsive landing page built using HTML, CSS, and JavaScript with an interactive fixed navigation menu that changes its style on scroll and hover.

## Features

- Responsive Design
- Fixed Navigation Bar
- Scroll Effect
- Hover Effects
- Modern UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Objective

To create a responsive landing page with an interactive navigation menu for an enhanced user experience.
